# Glycemic

Projenin amacı kullanıcıları Gıdaların glisemik değerlerine ulaştırmaktır. Kullanıcılarda adminler gibi gıda ekleyerek katkıda bulunabilir, gıdaları seçerek glisemik indeks hesaplaması yapabilirler.
